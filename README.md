# 🚨 Agentic AI Disaster Response Platform

An intelligent multi-agent disaster response coordination system built using Python, Streamlit, Google Gemini AI, and SQLite.

## Features

- 👤 Citizen Dashboard
- 🏥 Hospital Dashboard
- 🚑 Ambulance Dashboard
- 🏛 Government Dashboard
- 🤖 Multi-Agent AI Architecture
- 🛣 Road Intelligence
- 🏥 Hospital Allocation
- 🚑 Ambulance Allocation
- 📢 Emergency Alerts
- 📊 Live Analytics

## Tech Stack

- Python
- Streamlit
- Google Gemini API
- SQLite
- Plotly
- Folium
- Pydantic

## Project Structure

```
agents/
config/
dashboards/
database/
models/
orchestrator/
services/
utils/
```

## Run the Project

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Environment Variables

Create a `.env` file:

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```
